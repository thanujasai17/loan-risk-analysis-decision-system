# Loan Risk Analysis & Decision Support System

## Project Overview
This project analyzes bank loan applications to identify approval trends, applicant risk profiles, and business insights. The objective is to support data-driven lending decisions and reduce loan default risks.

## Problem Statement
Banks need an efficient way to assess loan applicants based on financial and historical data. This project provides analysis and visualization to improve decision-making.

## Tools Used
- MySQL
- Microsoft Excel
- Power BI

## Dataset
The dataset contains 614 loan applications with attributes such as:
- Gender
- Education
- Applicant Income
- Loan Amount
- Credit History
- Property Area
- Loan Status

## Project Workflow
1. Data Cleaning using SQL
2. Data Analysis using SQL Queries
3. Pivot Analysis in Excel
4. Business Insights Generation
5. Dashboard Creation in Power BI
6. Risk Categorization (Low / Medium / High)

## Key Analysis
- Loan approval vs rejection analysis
- Approval by gender
- Approval by education
- Property area distribution
- Risk distribution
- Approval rate KPI

## Risk Logic
- High Risk → No credit history
- Medium Risk → Low income applicants
- Low Risk → Good credit history and stable income

## Business Insights
- Applicants with positive credit history have higher approval rates.
- Graduate applicants show stronger approval trends.
- Urban applicants contribute the highest number of applications.
- High-risk applicants have lower approval rates.

## Business Recommendations
- Prioritize applicants with strong credit history.
- Use risk scoring before loan approval.
- Monitor high-risk applicants to reduce bad loans.
